# ValorHub - ฐานข้อมูล Valorant สุดล้ำ

ValorHub คือเว็บแอปพลิเคชันฐานข้อมูลเกม Valorant ที่เน้นประสิทธิภาพและความมินิมอล ออกแบบมาเพื่อให้ผู้เล่นเข้าถึงข้อมูลล่าสุดได้แบบ Real-time ผ่านอินเทอร์เฟซที่ผสานระบบ 3D เข้ากับความสวยงามที่ทันสมัย รองรับการใช้งานอย่างเต็มรูปแบบทั้งบนคอมพิวเตอร์และมือถือ

## คุณสมบัติเด่น

- **Dynamic Hero Section**: ส่วนแสดงผลหลักที่มีฉากเอเจนท์ 3D โฮโลแกรม พร้อมระบบตรวจจับเมาส์เพื่อสร้างมิติ Parallax
- **Agent Intelligence**: ข้อมูลเชิงลึกของเอเจนท์ทุกคน รวมถึงบทบาทและความสามารถพิเศษ
- **Weapon Arsenal**: รายการอาวุธ สถิติ และที่รวมสกินอาวุธจากทุกคอลเลกชัน
- **Collection Explorer**: ระบบเรียกดูบันเดิลและสกินใหม่ล่าสุดที่มีในตัวเกม
- **Interactive Map Previews**: มุมมองแผนที่เชิงยุทธศาสตร์จากทุกพื้นที่ในเกม
- **Global Background System**: ระบบพื้นหลังดวงดาวและอนุภาค 3D ที่ต่อเนื่องตลอดการเปลี่ยนหน้าเว็บ
- **Real-time Updates**: ระบบนับถอยหลังการอัปเดตแพตช์ถัดไปโดยประมาณ อ้างอิงจากรอบการอัปเดตของ Riot Games
- **Pagination Support**: ระบบแบ่งหน้าที่มีประสิทธิภาพสำหรับการเรียกดูข้อมูลจำนวนมาก เช่น เพลเยอร์การ์ด
- **Localization**: รองรับภาษาไทยและภาษาอังกฤษอย่างเต็มรูปแบบ (ผ่านข้อมูลจาก API)

## เทคโนโลยีที่ใช้

- **เฟรมเวิร์ก**: Next.js 15.0.3 (App Router)
- **ภาษา**: TypeScript
- **การจัดแต่งสไตล์**: Tailwind CSS
- **แอนิเมชัน**: Framer Motion
- **การแสดงผล 3D**: Three.js, @react-three/fiber, @react-three/drei
- **ไอคอน**: Lucide React
- **แหล่งข้อมูล**: Valorant-API.com

## การเริ่มต้นใช้งาน

ทำตามขั้นตอนเหล่านี้เพื่อรันโปรเจคในเครื่องของคุณ:

1. **Clone the repository** (คัดลอกไฟล์โปรเจค)
2. **Install dependencies** (ติดตั้งปลั๊กอินที่เกี่ยวข้อง):
   ```bash
   npm install
   ```
3. **Run the development server** (รันเซิร์ฟเวอร์จำลอง):
   ```bash
   npm run dev
   ```
4. **Build for production** (สร้างไฟล์เพื่อนำไปใช้งานจริง):
   ```bash
   npm run build
   ```

## โครงสร้างโปรเจค

- `/app`: รวบรวมหน้าเว็บและการตั้งค่าเลย์เอาต์ทั้งหมดของ Next.js
- `/components`: ส่วนประกอบ UI ที่นำกลับมาใช้ซ้ำได้ เช่น Navbar, HeroSection และพื้นหลังดวงดาว
- `/lib`: ฟังก์ชันช่วยเหลือและการเชื่อมต่อข้อมูลผ่าน API
- `/public`: ไฟล์สถิตและรูปภาพต่างๆ ที่ใช้ในแอปพลิเคชัน

## ปรัชญาการออกแบบ

ValorHub ยึดมั่นในหลักการ "Minimal, Seamless, Accurate" (เรียบง่าย, ไร้รอยต่อ, แม่นยำ) ทุกองค์ประกอบตั้งแต่ตัวอักษรไปจนถึงแอนิเมชันขนาดเล็กถูกสร้างขึ้นเพื่อให้ผู้ใช้ได้รับข้อมูลที่ต้องการอย่างรวดเร็วและสวยงามที่สุด

## เครดิต

ข้อมูลทั้งหมดสนับสนุนโดย Valorant-API โปรเจค ValorHub เป็นสิ่งที่สร้างขึ้นโดยแฟนคลับและไม่มีความเกี่ยวข้องอย่างเป็นทางการกับทาง Riot Games
